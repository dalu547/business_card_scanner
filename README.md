# Business Card Scanner Demo (Flutter)

This demo covers the core flow:
- scan card from camera
- or import from gallery
- extract text using ML Kit OCR
- segregate Name / Mobile Number / Address
- show saved items in list

## Run
```bash
flutter pub get
flutter run
```

## Notes
- This is a demo / POC
- Data is stored in memory only
- iOS and Android permission notes are included in separate text files
